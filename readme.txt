== Description ==
Create visual vehicle inspection reports with:
- Drag-and-drop condition markers
- Component progress bars
- Instant PNG report generation

== Installation ==
1. Upload the ZIP via WordPress admin
2. Activate the plugin
3. Access via "Inspection Report" in admin menu
